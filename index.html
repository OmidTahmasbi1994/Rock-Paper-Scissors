My name is Omid Tahmasebi Boldaji.
I'm studying computer engineering and this game was originally a student project, but later I decided to share it with you. This game is one of those games, especially small children are very interested in it.
The way flash cards move in this project is implemented with two different algorithms. Using "Array" and "Random Index".
Currently, the Random Index is working in this game, but the Array algorithm's code is also accessible on my Github page and it's based on JavaScript.
